# oh-my-termux

[中文](README_CN.md)

## Requirement

* Git
```bash
apt-get install -y git
```

## Installation

* Shell script
```bash
sh -c "$(curl -fsSL https://github.com/hh2333/oh-my-termux/raw/master/online-install.sh)"
```

* Git
```bash
git clone https://github.com/hh2333/oh-my-termux.git && cd oh-my-termux && ./install.sh
```
