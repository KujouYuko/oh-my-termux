# oh-my-termux

[English](README.md)

## 要求

* Git
```bash
apt-get install -y git
```

## 安装

* Shell 脚本
```bash
curl -fsSL https://github.com/hh2333/oh-my-termux/raw/master/online-install.sh | bash
```

* Git
```bash
git clone https://github.com/hh2333/oh-my-termux.git && cd oh-my-termux && ./install.sh
```
